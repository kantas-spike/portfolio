古い看板に使うようなフォントファミリーを教えて

  • #ChatGPT
  • #Font
2023/02/02
Q
昔の看板のような画像を作成したい。50年ぐらいの看板に使われているようフォントファミリーを教えてください。 もし、そのようなフォントファミリーの中で、open font licenseで公開されているフォントがあれば教えてください。

英訳

I want to create a graphic that looks like an old billboard.
Could you please tell me the font family like those used on billboards about 50 years ago?
If there are any fonts released under the open font license among such font families, please let me know.

先生の回答

A

Some popular font families used in billboards in the 1960s and 1970s include:

Futura Kabel Trade Gothic Neuzeit Akzidenz-Grotesk If you’re looking for open-source font alternatives:

Futura is available under the SIL Open Font License Trade Gothic is available under the Apache License You can find more information about these open-source font alternatives on websites like Google Fonts or GitHub.

調べる

ChatGPT先生が教えてくれたフォントは、いずれも、サンセリフ(Sans-serif)という書体でした。

サンセリフは、文字の「ひげ」部分がない書体で、日本ではゴシック体と呼ばれる種類のフォントのようです。

サンセリフにはいくつか分類があり、今回教えてくれたフォントは以下に分類されるようです。

フォント名 サンセリフの分類
Futura Geometric
Kabel Geometric
Trade Gothic Grotesque
Neuzeit Geometric と Neo-Grotesque のミックス
Akzidenz-Grotesk Grotesque

分類の説明は、Sans-serif - Wikipedia にありました。

まとめ

看板には、遠くからでも視認しやすいサンセリフのフォントが利用されやすいと理解しました。

また、今回奨められたフォントには、サンセリフの分類であるHumanistが含まれていませんでした。 これはHumanistが1976年以降にコンピューターモニターの表示用に発展してきたフォントであるため、古い看板には採用されていないのだと理解しました。

古い看板っぽくするには、Grotesque、Geometric、Neo-Grotesqueまたは、それらの組み合わせのフォントを採用すると良さそうです。

動画やWeb素材として使う場合は、オープンなライセンスのフォントを利用したいです。 今回紹介されたフォントはオープンなライセンスのフォントではないため、同様の分類のフォントを探す必要があります。

Making sense of typographic classifications – Fonts Knowledge - Google Fontsには、フォント分類と具体的なフォント名が記載されているので参考になりそうです。
